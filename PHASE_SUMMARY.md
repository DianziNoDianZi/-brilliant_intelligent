## 核心能力

- **LLM 规划**：Qwen-2.5 7B 将自然语言指令转为结构化按钮序列
- **WSG 动态检测**：UIAutomation 实时获取桌面 UI 元素
- **双回路**：慢回路（LLM）+ 快回路（技能库匹配），3 次积累后自动切换
- **内模拟**：系综世界模型在没有真实执行的前提下验证动作序列可行性
- **技能生成**：LLM 提案 + 内模拟验证，无需真实桌面交互
- **技能重组**：自动识别跨应用技能对，生成复合技能
- **冲突仲裁**：版本管理 + 降级机制 + 性能对比

## 当前局限

1. **记事本技能字符级生成** — batch 生成的记事本技能须使用 `type:TEXT` 格式
2. **跨应用桌面切换** — 技能重组中的 Alt+Tab 切换未全流程桌面验证
3. **CUDA 加速** — Python 3.14 无 CUDA 支持，Qwen 7B 跑在 CPU 上约 2 tok/s
4. **模板覆盖** — 当前 55+ 技能主要覆盖个位数运算，对多位数组合的覆盖率有限
